# Najczęściej używane foldery programu Soneta (enova365)

Poniższa lista zawiera ścieżki folderów (`programFolder`) wraz z opisami.
Używaj ich z funkcjami MCP `soneta_ui`: `navigate_to_folder`, `retrieve_list`, `open_form`.
